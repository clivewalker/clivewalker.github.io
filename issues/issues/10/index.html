Perch CollectionList fieldtype

A fieldtype for Perch Runway CMS to select an item from a collection. Similar to Dataselect, but for Perch Runway Collections rather than Perch regions.

github.com

How Do I?

How to add an image class to all images uploaded via Redactor editor

In our case, we wanted all images uploaded into Redactor to have the Bootstrap class img-responsive applied. This function is referenced in the Redactor documentation under Image Callbacks. You need to jam a bit of code into the Redactor settings in _config.inc.

thebigblueswim.com

Penner Builders - Custom RTM Solution

Perch was used for the generation of each page. The gallery uses the Perch Gallery app, plus Blue Imp Gallery (for controls). The House Plans page uses multiple items with perch_content_custom and filters to spread them across the tabs. On the RTM Specs page, each list and heading is a block. The contact form is Perch Forms add-on. The Testimonials at the bottom of each page is a shared region using a repeater.
