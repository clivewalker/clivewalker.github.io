Tips And Tricks

Quick Tip: Image Previews with SimpleMDE Editor

SimpleMDE editor ships with Perch and it has a useful side-by-side live preview mode. This means that images added via Perch, which appear as shortcodes in the editor window, can be displayed inline as you type. Nice.
