News

Perch Apps: Adding functionality to your site

An overview of the benefits of Perch apps. And a description of several custom apps including an Events app and a web forms app.

The Events app lets users view their current events in either a calendar, daily, monthly or yearly views. Events can be generated for multiple dates if you have an event that repeats often.

The web forms app allows our client to build large custom web forms, using simple form inputs, file uploads right through to even allowing them to take payments.
