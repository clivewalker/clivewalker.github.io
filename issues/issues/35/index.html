Issue 35  11th January 2018

Introduction

Happy New Year Everyone!

I hope you had a great holiday season. I spent part of it staying in my friends' hotel in Cornwall on the rugged south west coast of the UK. It was a fab break!

In this issue, there's a new app for styling Perch admin and a useful, simple, Redactor customisation config file. There have been quite a few posts about Redactor editor customisation on the Perch Forum so this config file might be just what you need.

Also, if you missed it last time, here's a link to join Perchology Slack which is a community chat for Perchers. It's a replacement for the Perch Chat Slack which closed at the end of last year.

See you next time.

Clive Walker - Perchology
