Add-Ons

Perch Image Optimiser

The Perch Image Optimiser will compress images that are generated by the CMS.

This new add-on by Red Finch (James Wigger) looks very useful. Settings to enable or disable optimisation tools, set the compression amount, strip meta data, enable or disable SVG optimisation. Scheduled Tasks are used.

github.com

Perch Shop now in Chirp SEO

The latest release of the Chirp SEO add-on now supports Perch Shop. It already allows you to analyse the content of pages, blog posts and collections. Now, it can check your product pages as well. Great work by Ryan Gittings.

grabachirp.com

Perch Event Logger

This add-on tracks the activity of users with access to your CMS.
