Keyboard Shortcuts

Turn on editing and keyboard shortcuts In the Perch settings page. Navigate round the Perch admin using shortcuts like, for example g + p for Page Listing and g + d + r for Diagnostics Report.

Use Ctrl/Cmd + S to Save a Region after editing.

Use Ctrl/Cmd + E to edit a page. Needs perch_get_javascript(); and after that browse your site and click Ctrl/Cmd + E to jump to the edit screen for that page.
