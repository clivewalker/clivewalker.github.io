Sendero Drilling Rig Locator

This site's rig locator page uses a customisable Google maps implementation. The client enters location details using simple text fields. The data is used in the JavaScript code for the map. It's a good example of how Perch can be used for more than just HTML templates. Nice work by Robert Hughes.

Perch Editing Shortcuts

Did you know? You can use Cmd/Ctrl + S to Save the content region you are editing. And Ctrl + E can be enabled to jump into editing the content for the page you are viewing.
