Introduction

Hello again

I've been working with an existing Perch site this week by adding a product section with categories. As a result, I've been reading up on Perch Categories, there's a link below. After starting the work, I realised that I'd developed a similar system on one of my personal sites and that was also a real help on the new project.

Personal sites might seem like a diversion sometimes but they are incredibly useful. I've always been able to re-use code and transfer knowledge to other sites from personal projects. And Perch and its template system is well-suited to that.

Anyway, hope you like these links. See you next time.
