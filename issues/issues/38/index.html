Add-Ons

Edit Regions from Front-end

With standard Perch you can add the functionality to edit pages with Ctrl-E. You have to click to the correct region type and then to the correct region to edit it.

This solution makes it possible for administrators to edit a region directly from the front-end of a website. You go to the page you want to modify, you type Shift-Alt-E to show editable items on the page. Click one of the edit buttons to go directly to the region edit screen.

Nice work!
