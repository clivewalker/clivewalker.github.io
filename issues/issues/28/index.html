Introduction

Hi folks

Perch fact: Did you know? It's possible to use the old English word 'betwixt' instead of 'between' for the match attribute in Perch conditionals. I rediscovered this recently when browsing the Perch core code (as you do). I think this is a 'hidden feature' :-)

Perch Documentation

Correcting Links with the replace attribute

How do I make sure people enter correct links to a website address? Content editors sometimes forget to add a full website address including the http:// leaving you with broken links on the site. You can use the replace attribute on a template tag to ensure that you end up with an http:// at the beginning of a URL

grabaperch.com

See what is available in a template

The <perch:showall />tag is incredibly useful for displaying all available fields in a template (including Perch-generated fields). I used it last week to find out how to order items in a multiple item region when there was no immediately obvious field to order by.
