Add-Ons

InputMask Text FieldType

A field type to add an input mask to a Perch text field. An input mask helps the user with the input by ensuring a predefined format. For example, for dates and phone numbers. Uses the input mask library by Robin Herbots .

gitlab.com

Tips and Tricks

The Append Attribute

A handy Perch template feature is the append attribute. The template engine appends the value of the append attribute to the end of a field. Perhaps it is most commonly used when you want to append an ellipsis to truncated text.

grabapipit.com
