From The Forum

Importing Members

It's possible to import Members directly into the database. But, check that your data in the memberProperties column is complete. Blank values can give an error message when sorting columns in Perch.

Run and test any such imports on your testing server first!

community.perchcms.com

Other

Running a local server with LocalWP

As an alternative to MAMP or other local development environments, you might like to consider LocalWP which is a WordPress development tool that you can download and run on your machine. Huh? WordPress development?! How does that work? 

Well, creating a local site with LocalWP is just a few clicks and it will install WordPress. However, it's easy to delete the WordPress files and amend the database to run non-WP sites.

It's possible to keep MAMP and try LocalWP on the same machine. I've tried LocalWP and it's a smooth experience.

Good tip from Wayne Hooper.

localwp.com
