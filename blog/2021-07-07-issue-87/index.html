The Newsletter has closed. But, here's one of the Issues

Issue 87

07 July 2021 by Clive Walker

Hey folks

There's some exciting news this week with an update on Perch 4. A preview release! It's great to know that this is almost ready.

In other news, I've been scoping a new project that will use the Members app together with two third party add-ons to create a grant application management portal for a client. It could be a really exciting job and demonstrates that Perch is very extensible. In fact, a lot of great work has been done to extend Perch. If you want to find out more, the Perchology Links repo on Github has a list of third party add-ons and is well worth browsing!

That's it for now. See you next time.

Clive Walker - Perchology
