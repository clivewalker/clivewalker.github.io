The else Attribute

How to use the else attribute without using <perch:if></perch:if> tags. Really useful for specifying a default image, for example.

docs.grabaperch.com

The no-index Attribute

The no-index attribute tells Perch not to index a field. In general, any field that is not needed for sorting or filtering does not need to be indexed. The attribute works for content regions and collections.

docs.grabaperch.com
